# Project Overview

This is a Java Spring Boot project. The `pom.xml` file indicates that it uses Spring Boot version 3.4.0 and Java 21. The project is a simple web application, as indicated by the `spring-boot-starter-web` dependency.

# Building and Running

*   **Build:** `mvn clean package`
*   **Run:** `mvn spring-boot:run`
*   **Test:** `mvn test`

# Development Conventions

*   The project follows the standard Maven project structure.
*   The main application class is `com.minyk.test.lazybird.TestLazyBird`.
